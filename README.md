# ezForge Community

Welcome to the ezForge community! This repository hosts our public **GitHub Discussions** for questions, feature requests, bug reports, and community showcases.

## Discussion Categories

| Category | Use It For |
|----------|------------|
| **Q&A** | Ask questions about ezForge Platform, Ready, or Hub |
| **Feature Requests** | Suggest new features and improvements |
| **Show & Tell** | Share your MCP servers, integrations, and projects |
| **Bug Reports** | Report bugs or unexpected behavior |
| **Announcements** | Product updates and releases from the ezForge team |

## Guidelines

- **Be respectful.** Treat everyone with kindness and professionalism.
- **Search first.** Check existing discussions before creating a new one.
- **Stay on topic.** Use the appropriate category for your post.
- **No sensitive data.** Don't share API keys, passwords, or personal information.
- **Security issues** — Please email `security@ezforge.ai` instead of posting publicly.
